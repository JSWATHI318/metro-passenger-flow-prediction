# Metro Passenger Flow Prediction using Deep Learning

This project was developed as part of a 4-member academic group project.

The system predicts metro passenger flow using Machine Learning and Deep Learning models such as MLP, GRU, and LSTM.

## Technologies Used

* Python
* Django
* TensorFlow/Keras
* Scikit-learn
* Pandas
* NumPy
* Matplotlib

## Features

* Passenger flow prediction
* Dataset preprocessing
* Deep Learning model training
* Graph visualization
* RMSE and MAE evaluation

## How to Run

Install requirements:

pip install -r requirements.txt

Run server:

python manage.py runserver

Open browser:

http://127.0.0.1:8000/index.html
